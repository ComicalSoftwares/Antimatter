Incredibly stable, blazing fast, and light on resources.

Note: Recommended for non-important projects during alpha. Expect a slight learning curve for the streamlined workflow.

Key Features

High Performance

Minimalist design favorises speed. Light on CPU and memory usage.

Discord Presence

Built-in pypresence support to show off what you're coding.

Auto-VENV

Automates repetitive tasks like creating and activating .venv environments.

Built-in Terminal

Non-interactive terminal perfect for quick debugging and logs.

One-Click Compile

Easily build applications using PyInstaller directly from the IDE.

Pyflakes Integration

Built-in issue checking to catch bugs before you run your code.
